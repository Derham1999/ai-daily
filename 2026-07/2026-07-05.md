### 产品与功能更新

1. **Claude Code 技能生态迎来爆发式增长。**
   [JuliusBrussee/caveman](https://github.com/JuliusBrussee/caveman) 以"原始人"语言风格实现 **65% token 削减**，日增 **1089 星**突破 8.3 万总星。同时，[mattpocock/skills](https://github.com/mattpocock/skills) 发布 15 万星技能库，[alirezarezvani/claude-skills](https://github.com/alirezarezvani/claude-skills) 提供 **337 个技能插件**覆盖工程、营销、产品等多场景。

2. **OpenAI 正式发布 Codex-Claude Code 桥接插件。**
   [openai/codex-plugin-cc](https://github.com/openai/codex-plugin-cc) 允许开发者从 Claude Code 调用 Codex 进行代码审查或任务委托，日增 **718 星**，总星突破 **2.4 万**。标志着主流 AI 编码工具互操作性迈出关键一步。

3. **阿里巴巴开源 Page Agent，自然语言控制 Web 界面。**
   [alibaba/page-agent](https://github.com/alibaba/page-agent) 实现浏览器内 GUI 代理，通过自然语言指令操控网页界面，日增 **742 星**，总星达 **2.3 万**。降低 Web 自动化门槛，拓展 AI Agent 应用边界。

4. **Chrome DevTools MCP 桥接编码代理与浏览器调试。**
   [ChromeDevTools/chrome-devtools-mcp](https://github.com/ChromeDevTools/chrome-devtools-mcp) 为 AI 编码代理提供浏览器 DevTools 能力，支持性能分析、网络监控等调试功能，总星突破 **4.5 万**。
