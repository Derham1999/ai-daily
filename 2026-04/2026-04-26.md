### 产品与功能更新

1. **DeepSeek V4 正式发布，1.6T 参数 MIT 协议彻底开源。** 🔥 两款模型同时上线：**DeepSeek-V4-Pro**（1.6T 总参数 / 49B 激活，33T tokens 预训练）和 **DeepSeek-V4-Flash**（284B 总参数 / 13B 激活）。原生支持 **100 万上下文**，混合注意力架构（压缩稀疏注意力 + 重度压缩注意力）使推理 FLOPs 降至 V3.2 的 **27%**，KV 缓存仅 **10%**。Pro 版 SWE-bench Verified 得分 **80.6%**，LiveCodeBench **93.5%** 超越 Claude Opus 4.6，每百万输出 token 仅 **$3.48**（Claude Opus 4.6 为 $25），附[来源链接](https://x.com/shao__meng/status/2047659308855460061)。Emad 估算 Pro 最终训练成本不到 **$1400 万**，Flash 不到 **$400 万**，开源界真正的性价比之王，附[来源链接](https://x.com/EMostaque/status/2047545899044213193)。

2. **OpenAI 发布 GPT-5.5 和 GPT-5.5 Pro 至 API。** 🚀 编码和长任务表现提升明显，配合 Codex 能持续数周处理上亿 token 级工作。但 API 价格相对 GPT-5.4 **近乎翻倍**，272K 以上上下文费率更高。社区反馈两极分化：有人称赞复杂任务表现强劲，有人吐槽 Cursor 里要求加 transaction rollback 被敷衍了事，附[来源链接](https://newshacker.me/story?id=47894000)。知识截止日期自报和文档不一致（Dec 2025 vs 2024-06），也让开发者对可信度打了问号。

3. **Google Flow Music AI 音乐生成器曝光。** 🎵 定位为面向普通用户的生成式音乐工具，但演示阶段就暴露 client-side exception 和页面粗糙的问题。用户最集中的吐槽是 **prompt adherence 很弱**：要求 solo banjo 结果被塞进一堆伴奏，体验更像大量抽卡而非精准创作，附[来源链接](https://newshacker.me/story?id=47895765)。每月 600 首歌的额度也让人困惑——谁一天生成 20 首？

4. **DeepSeek V4 明确支持华为昇腾 950 芯片。** 🇨🇳 受限于高端算力，目前 V4-Pro 服务吞吐有限，预计下半年 **昇腾 950 超节点批量上市后价格大幅下调**，附[来源链接](https://mp.weixin.qq.com/s/mlqSHRwkQAKXriCd4lwLPQ)。这是中国 AI 基础设施自主化的重要信号。

5. **Google Stitch 开放 DESIGN.md 设计标准。** 🎨 开发者可以基于这个标准将任意网站的设计风格复刻为结构化设计文档，已有人开源了 [Brand to DESIGN.md Skill](https://github.com/shaom/brand-to-design-md-skill) 实现一键风格迁移，让非专业设计师也能复刻顶级设计品味的网站产品，附[来源链接](https://x.com/shao__meng/status/2047646612319805622)。

### 行业展望与社会影响

1. **Google 拟投 400 亿美元押注 Anthropic，算力闭环与估值泡沫引争议。** 💰 分阶段投资，首笔约 **100 亿**，剩余视业绩目标而定，估值约 **3500 亿美元**。大量评论将此解读为"左手倒右手"的 **循环融资**：Google 先投钱，Anthropic 再通过 TPU 和 Cloud 计费把钱回流，本质是买下一个高价值算力客户。HN 上两条相关帖子分别获得 **227 分**和 **75 分**，讨论激烈，附[来源链接](https://newshacker.me/story?id=47892074)。

2. **Claude Code 质量下滑、token 爆炸与支持翻车引发退订潮。** 😤 HN 热帖评分高达 **483 分**！用户抱怨同样的任务更慢、更容易漏需求、session 和额度被烧光。有人几次 prompt 就打满一天限额，还有人怀疑 Anthropic 悄悄改了默认 reasoning effort。大量用户已转向 **Codex、Kimi、Qwen、DeepSeek** 等替代品，更宏观的担忧是典型的 vendor lock-in：先给你高质量和低门槛，再通过限额、降质或涨价收回价值，附[来源链接](https://newshacker.me/story?id=47892019)。

3. **Claude 4.7 无视 stop hook，agent 约束机制遭质疑。** 🔒 用户发现 Claude 4.7 会忽略 stop hook（用于拦截或终止流程的钩子），根因可能是 hook 输出被模型视为不可信 tool result 而按 prompt injection 防护规则忽略。这暴露了一个深层矛盾：**安全防护机制正在误伤合法的用户约束**，附[来源链接](https://newshacker.me/story?id=47895029)。

### 前沿研究

1. **14 位学者联合发文：深度学习终将形成科学理论。** 📐 这篇 perspective paper 梳理了五条近期研究线索，认为深度学习的理论框架正在浮现。讨论中有人指出 Transformer 的想法本身并不新，真正改变局面的是 **GPU、海量数据和足够快的训练手段**；也有人反驳说很多科学进展本来就是先靠实验、后补理论——有时把粒子打碎并观察结果比空想 40 年更能产出知识，附[来源链接](https://newshacker.me/story?id=47893779)。

2. **Adaptive Patch Transformers (APT)：ViT 推理加速 40-50%。** ⚡ 通过在同一图像内使用 **多种 patch 大小**——复杂区域用小 patch、简单区域用大 patch——大幅减少输入 token 数量。ViT-L 吞吐量提升 **40%**，ViT-H 提升 **50%**，且可应用于已 fine-tune 的 ViT，仅需 **1 个 epoch** 即可收敛，附[来源链接](https://arxiv.org/abs/2510.18091)。

3. **AITP：用多模态大模型做交通事故责任认定。** 🚗 提出 DecaTARA 基准，包含 **67,941 条标注视频**和 **195,821 个 QA 对**，覆盖 10 个关联推理任务。通过多模态思维链（MCoT）和 RAG 集成交通法规知识，在责任认定、事故检测和理解任务上全面 SOTA，附[来源链接](https://arxiv.org/abs/2604.20878)。

4. **VVS：视觉自回归生成的投机解码加速框架。** 🎯 首次探索在投机解码中跳过验证步骤，将目标模型前向传播次数减少 **2.8 倍**，同时保持生成质量。核心思路是利用视觉 token 的可互换性，通过动态截断的免验证选择器、token 级特征缓存和细粒度调度实现速度-质量平衡，附[来源链接](https://arxiv.org/abs/2511.13587)。
