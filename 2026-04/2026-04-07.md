### 产品与功能更新

### 前沿研究

1. **北大团队改造 DeepSeek 注意力机制 HISA**，速度快四倍几乎不丢精度
   提出**分层索引稀疏注意力（HISA）**，突破 64K 上下文的索引瓶颈，相比 DeepSeek 正在使用的 DSA 提速 **2-4 倍**，即插即用无需重新训练。论文直接在 DeepSeek-V3.2 上验证有效 🔥 [来源](https://www.qbitai.com/2026/04/396863.html)

2. **ReCALL 框架化解生成式与判别式的范式冲突**｜CVPR'26
   独创"诊断-生成-校准"闭环体系，让多模态大模型在组合图像检索任务上全面超越 SOTA，解决了生成式模型被强行改造为判别式检索器后出现的能力退化问题 🎯 [来源](https://www.qbitai.com/2026/04/396863.html)

3. **苹果研究证明 AI 模型连小学数学都做不好**
   研究者对 GSM8K 基准测试进行变体测试，发现 AI 只是在做模式匹配，遇到意外情况就安静地给出错误答案。Yann LeCun 转发评论引发广泛讨论 🧮 [来源](https://x.com/ylecun/status/2041295545495380356)

### 社媒观点

1. **文件系统比 RAG 和 MCP 更懂 AI Agent**
   开发者指出代码幻觉是数据问题而非模型问题——API 每天变更但模型知识停留在过去。将文档网站挂载为文件系统，遵循 Unix"一切皆文件"哲学，可能比 RAG 分块和 MCP JSON Schema 更靠谱 📁 [来源](https://x.com/shao__meng/status/2041330844481053003)

2. **Anthropic 增长团队分享 AI 增长营销四个维度**
   从自动化手动流程 → 借力 AI 认知优势 → 重构工作流 → 创造全新品类，按价值层级递进。关键是"用 AI 扩大思维覆盖面"而非仅仅加速现有工作 📈 [来源](https://x.com/shao__meng/status/2041309262350065667)

3. **从"想法"到"可运行软件"的四步闭环**
   grill-me 需求澄清 → write-a-prd 规范固化 → tdd 增量实现 → agent-browser 端到端验证，四个 Skills 组成可靠交付流水线 🔧 [来源](https://x.com/shao__meng/status/2041312035250586056)

4. **Jitendra Malik 呼吁机器人操作标准化基准**
   建议使用 STT（Success weighted by normalized inverse Task Time）作为标准化指标，取代目前满大街的 cherry-picked demo。Yann LeCun 转发支持 🤖 [来源](https://x.com/ylecun/status/2041290168959631869)

5. **"认知投降"导致 AI 用户放弃逻辑思考**
   研究发现过度依赖 AI 会导致用户出现"认知投降"现象，放弃独立逻辑思考。AI 工具越强大，人类越需要保持警惕 🧠 [来源](https://www.reddit.com/r/artificial/comments/1se2nxm/cognitive_surrender_leads_ai_users_to_abandon/)

6. **Reducto 发布 Deep Extract**：用 Agent-in-the-Loop 替代 Human-in-the-Loop
   针对长文档高复杂度场景，通过 Extract → Verify → Correct 三阶段自主验证循环，让 Agent 自己交叉比对、迭代修正 📄 [来源](https://x.com/shao__meng/status/2041326435978129801)
