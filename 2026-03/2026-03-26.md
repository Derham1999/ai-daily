### 行业展望与社会影响

1. **“Proof of Human” 正在从概念口号变成平台刚需。**
   围绕一篇关于 **Proof of Human** 的讨论，Marc Andreessen 等人强调，随着 agentic capability 快速提升，互联网平台将越来越需要区分“真人”和“自动体”。[来源链接](https://x.com/pmarca/status/2036880088357380465) 这背后其实是在提醒行业：下一轮竞争不只是更强的 AI，还有 **身份验证、信任基础设施与反滥用体系**。

2. **AI 编码的争论已经从“能不能用”变成“该不该踩刹车”。**
   Hacker News 讨论聚焦在 AI 代码直接进生产、coding agent 审核边界以及开发流程要不要放慢。[来源链接](https://newshacker.me/story?id=47517539) 风向很现实：行业不再沉迷“自动化神话”，而是在追问 **责任归属、人工复核与交付质量**，这才像成熟市场会问的问题。

3. **从零搭建 coding agent 的经验正在外溢成工程共识。**
   另一篇讨论指出，真正难的不是调模型，而是 **状态管理、上下文压缩、任务结束条件** 这些工程细节。[来源链接](https://newshacker.me/story?id=47515605) 这说明 coding agent 赛道已经进入“拼系统设计”的阶段，单靠模型能力堆料开始不够看了。

### 前沿研究

1. **机器人也开始需要专门的 coding agent 基准了。**
   论文 **CaP-X** 提出面向机器人操作的 Code-as-Policy 研究框架与基准，覆盖 **12 个模型**，结果显示：性能会随着人工设计抽象减少而明显下降，但可通过多轮交互、执行反馈、视觉差分和技能合成把可靠性拉回来。[来源链接](https://arxiv.org/abs/2603.22435) 这很关键——机器人 agent 的上限，不只看模型聪不聪明，也看能不能在真实反馈中学会补救。

2. **推理时扩展不一定非得烧大算力。**
   论文 **Tiny Inference-Time Scaling with Latent Verifiers** 提出直接在扩散模型隐藏状态上做验证，避免像素空间来回解码与重编码。[来源链接](https://arxiv.org/abs/2603.22492) 作者报告称，在相同预算下可把联合生成与验证时间降低 **63.3%**、FLOPs 降低 **51%**，同时在 GenEval 上还有 **+2.7%** 提升，属于那种“既省钱又涨分”的正经优化。

3. **视频生成开始补“长期一致性”这门旧债。**
   新论文 **I3DM** 试图通过 **3D-aware memory retrieval and injection** 解决视频生成在重访场景时容易穿帮的问题。[来源链接](https://arxiv.org/abs/2603.23413) 另一篇 **RealMaster** 则把渲染场景提升为更写实的视频输出。[来源链接](https://arxiv.org/abs/2603.23462) 很明显，视频模型的战场正在从“第一眼惊艳”转向 **连续镜头、空间一致性与可控性**。
